# /brain-memo

The canonical, deployable version of this command lives in `project-seed/.claude/commands/brain-memo.md` at the root of this tooling repo. It is host-agnostic (GitLab first, GitHub fallback, local draft as last resort) and reads the brain location from `BRAIN_REPO_PATH`.

Deploy it from the seed pack rather than from here, so every project carries the same version. See `project-seed/README.md` for how the seed goes into the GitLab project template and existing repos.
